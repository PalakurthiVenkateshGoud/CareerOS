# CareerOS — Multi-Agent Career Intelligence Platform

> Your AI-Powered Multi-Agent Career Intelligence Platform

CareerOS transforms a resume, target role, and weekly time availability into
a complete career strategy — generated by 9 collaborating AI agents with
transparent, step-by-step reasoning.

## Architecture

User → Next.js Frontend → FastAPI Backend → CareerOS Orchestrator

→ [Resume, Role Research, Skill Gap, Career Twin, Strategy,

Roadmap, Project, Interview, Readiness] Agents

→ Microsoft AI Foundry / Azure OpenAI

→ Response Aggregator → Career Dashboard

## Tech Stack

- **Frontend:** Next.js, TypeScript, Chakra UI, Recharts, Lucide Icons
- **Backend:** FastAPI, Python
- **AI Layer:** Microsoft AI Foundry / Azure OpenAI
- **Resume Parsing:** PyMuPDF, pdfplumber
- **Database:** SQLite

## Quick Start

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install fastapi uvicorn pydantic-settings openai PyMuPDF pdfplumber
cp .env.example .env        # then fill in Azure OpenAI credentials
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
cp .env.local.example .env.local
npm run dev
```

Visit `http://localhost:3000`.

## Agents

| Agent | Role |
|---|---|
| Resume Intelligence Agent | Parses resume into structured profile |
| Role Research Agent | Builds a role blueprint for the target role |
| Skill Gap Analysis Agent | Compares profile vs blueprint |
| Career Twin Agent | Generates Current You vs Future You |
| Career Strategy Agent | Plans milestones |
| Learning Roadmap Agent | Generates week-by-week plan |
| Project Recommendation Agent | Suggests portfolio projects |
| Interview Coach Agent | Generates interview prep pack |
| Career Readiness Agent | Computes readiness score + projections |

## Safety

Every agent output includes `confidence`, `reasoning`, `assumptions`, and
`missing_data` fields, surfaced in the Agent Reasoning Trace.
